## 📌 2. 자바와 가상머신

- ***한 번만 작성하면 어디서든 동작한다 (Wrtie Once, Run Anywhere)***
- 자바는 `호환성` 문제를 해결해준다 ▶︎ JVM만 설치되면 어느 운영체제이든 어느 디바이스이든 동일하게 동작한다
- JVM : Java Virtual Machine
- JRE : Java Runtime Environment
- JDK : Java Development Kit
